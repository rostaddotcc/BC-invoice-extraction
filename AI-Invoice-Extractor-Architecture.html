🤖 AI Invoice Extractor

System Architecture & Technical Overview

Version 1.0.0.21

📋 Workflow

1

Batch Upload

The user uploads invoice images (PNG, JPG) via drag-and-drop. The system creates Import Document records with status "Pending".

⬇️
2

AI Processing (Qwen-VL)

Batch Processing Manager handles up to 3 concurrent calls. Images are converted to Base64 and sent to Qwen-VL API for OCR analysis.

⬇️
3

Data Extraction & Vendor Match

The AI returns JSON with invoice data. The system automatically matches vendor based on name/number and sets default GL account.

⬇️
4

Preview & Edit

The user reviews extracted data in the Invoice Preview page. The original image is displayed in the FactBox. All fields can be edited before creation.

⬇️
5

Create Purchase Invoice

The system creates a standard Purchase Invoice in Business Central with all lines and accounts pre-filled.

🏗️ Business Central (AL)

  • Tables 4 (Setup, Temp Buffer, Import Header/Line)
  • Pages 6 (Setup, Preview, List, Upload)
  • Codeunits 4 (API, Extraction, Batch Mgt, Worker)
  • BC Version 2024 Wave 2 (v27.4+)

🧠 AI / ML Stack

  • Qwen-VL Alibaba Cloud Vision-Language Model
  • API Endpoint dashscope.aliyuncs.com
  • Model qwen-vl-max
  • Output Strukturerad JSON

🔧 Integration & Data

  • HTTP Client AL HttpClient för API-anrop
  • Media Management BC Media Set för bildlagring
  • JSON Parsing AL JsonObject/JsonArray
  • Base64 Encoding Bildkonvertering för API

⚡ Performance & Security

  • Concurrency Max 3 concurrent processes
  • Timeout Configurable (default 60s)
  • API Key Masked in UI, encrypted storage
  • Queue Management Automatic retry on errors

🔄 Data Flow Between Components

User Upload → Import Document Header (Pending)
                    ↓
Batch Processing Mgt → Queue Management
                    ↓
Batch API Worker → Qwen VL API Codeunit
                    ↓
HTTP POST (Base64 Image) → Qwen-VL API
                    ↓
JSON Response → Invoice Extraction Codeunit
                    ↓
Parse & Save → Import Document Header/Line
                    ↓
User Preview → Invoice Preview Page
                    ↓
Create Invoice → Purchase Header/Line (BC Standard)
